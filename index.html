InstCov README

Introduction

InstCov is a C/C++ code instrumentation tool based on Clang. It's design is targeted at general-purpose instrumentation for coverage analysis and runtime verification. Currently, InstCov has implemented function coverage, decision/condition coverage and MC/DC coverage.
